# Bike_Project

Bike Sales Analysis – SQL, Excel, and Power BI (Beginner Project)

This project analyzes a Bike Sales dataset using three tools:
MySQL, Excel, and Power BI.

The goal is to show how one dataset can be cleaned, explored, and visualized across different tools used by Data Analysts.

This is a beginner-friendly project, with simple steps, clear logic, and easy-to-read dashboards.

**1. MySQL – Data Cleaning + Analysis**

In the SQL section, I cleaned the dataset, removed duplicates, and created basic exploratory queries.

**Main SQL steps:**

Checked for duplicates using window functions

Created a clean table (sales1)

Summarized customer, product, and revenue data

Found top categories, sub-categories, and profitable years

Calculated monthly revenue and running totals

Full details inside: **/MySQL/README.md**


**2. Excel – Beginner Dashboard**

In Excel, I created a simple, clean dashboard using PivotTables, PivotCharts, and slicers.

**Dashboard shows:**

Sales by Age Group

Sales by Gender

Revenue by Category

Monthly revenue trend

Top sub-categories

Full details inside: **/Excel/README.md**


**3. Power BI – Interactive Dashboard**

In Power BI, I built a more visual and interactive dashboard with drilldowns, slicers, and trend charts.

**Dashboard includes:**

Customer overview

Product profitability

Top categories and sub-categories

Yearly and monthly revenue

Running totals

Full details inside: **/PowerBI/README.md**
